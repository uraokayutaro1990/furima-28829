# Usage
まずは最初の画面で、商品一覧をみる
欲しいものや売りたいものがあれば、
名前、アドレス、生年月日を入力し、
パスワード設定を行い新規登録し、
ログイン後、購入、出品ができる。
出品する際は、商品のカテゴリ、値段、状態を入力。
 
DEMOの実行方法など、"furima-28829"の基本的な使い方を説明する
 
```bash
git clone  https://furima-28829.herokuapp.com
cd examples

```
